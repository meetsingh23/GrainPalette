About the Rice Type Classification Model

This model is built using Convolutional Neural Network and trained on labeled images of rice. It takes rice grain images as input and predicts the type accurately.

Accuracy of the Model

The model achieves around 97% accuracy.

Different Types of Rice

It can classify multiple rice categories.

Dataset Used

Trained on labeled rice image dataset.

Technical Architecture

Based on Deep Learning (CNN).
